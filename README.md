# Electron 简历平台

## 技术栈

React、Electron、TypeScript、Redux


### 下载
```
npm install electron --ELECTRON_MIRROR="http://npm.taobao.org/mirrors/electron/"
```

## 如何启动

```
yarn install

yarn dev

yarn start
```
